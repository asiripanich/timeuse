# timeuse

<!-- badges: start -->
<!-- badges: end -->

An R package that contains time use activity classifications. See a list
of known time use surveys
[here](https://unstats.un.org/unsd/gender/timeuse/index.html).

## Installation

``` r
library(remotes)
install_github("asiripanich/timeuse")
```
